# mbti-cli Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Initialize `mbti-cli` as a small, maintainable Go CLI with Cobra, unit tests, linting, build automation, and Apache 2.0 licensing.

**Architecture:** The binary entry point stays in `main.go`, Cobra commands live under `cmd/mbti-cli`, reusable helpers stay in `internal/cli`, and build metadata lives in `internal/version`. Tooling stays intentionally small: a `Makefile`, `golangci-lint`, `testify`, and a minimal README.

**Tech Stack:** Go 1.24, Cobra, Testify, golangci-lint, Make
